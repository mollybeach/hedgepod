## Environment Variables

Create `.env.local`:

```
NEXT_PUBLIC_PRIVY_APP_ID=your_privy_app_id
NEXT_PUBLIC_WORLD_APP_ID=your_world_app_id
NEXT_PUBLIC_ALCHEMY_API_KEY=your_alchemy_key
```
